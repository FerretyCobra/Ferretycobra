## 🚀 Featured Projects

### 🦠 [Outbreak Intelligence System](https://github.com/FerretyCobra/Outbreak-intelligence-system)
> Real-time Indian RSS disease signal detection · Python · FastAPI · MySQL · HuggingFace · Leaflet.js

- **6-stage fast-failure pipeline**: rule-based detection → Semantic Guard (MiniLM) → Zero-Shot NLI (distilbart-mnli) — filters ~90% of articles before ML inference runs
- **Multi-factor severity scoring** with 12 configurable thresholds, zoonotic bumps, growth velocity, and recency burst
- Hourly aggregation buckets for O(hours) spike detection, connection pooling, human review queue, CI pipeline

> Real-time sound level monitoring · Java · Android · PHP · MySQL · Leaflet.js

- Background foreground service + BootReceiver + home screen widget + Quick Settings tile
- Geospatial web dashboard with bounding-box filtering and hotspot detection for clusters >80 dB
